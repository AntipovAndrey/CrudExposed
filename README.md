# CrudExposed

Simple CRUD application to try out [Exposed](https://github.com/JetBrains/Exposed) ORM framework.
This REST service provides API to create documents with assigned tags.

This sample uses both [DSL](https://github.com/JetBrains/Exposed/wiki/DSL) and [DAO](https://github.com/JetBrains/Exposed/wiki/DAO) approaches.
